# Wrench CLI

Wrench is a command-line project manager built in Python using SQLite.

This project was created as a hands-on study of backend development fundamentals, focusing on building real-world concepts without frameworks.

---

## 🚀 What this project does

Wrench allows you to manage projects directly from the terminal:

- Create new projects
- List all projects
- Search projects by ID
- Update project status
- Delete projects

All data is persisted using SQLite.

---

## 🧠 What I learned building this

This project was developed as a learning journey into backend development.  
Key concepts practiced:

- CRUD operations with SQLite
- Repository pattern (data separation layer)
- Clean architecture (CLI / Database / Repository separation)
- Safe SQL queries using parameterized statements (SQL injection protection)
- Structuring a Python CLI application from scratch

---

## 🛠️ Tech Stack

- Python 3
- SQLite
- Standard Library only (no frameworks)

---

## 🔐 Security

This project uses parameterized SQL queries to prevent SQL injection vulnerabilities.
